# beeware-demo
Simple beeware example

### Set up environment

```
python3 -m venv beeware-venv
source beeware-venv/bin/activate

python3 -m pip install --upgrade pip

pip3 install -r requirements.txt
```

### Initialise project

```
briefcase new
```

### Run project

```
cd firstapp
briefcase dev
```
